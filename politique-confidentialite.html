Politique de confidentialité

Dernière mise à jour : 2026

Introduction

Garoms-tech JBD-X accorde une grande importance à la protection des données personnelles des utilisateurs de son site internet.

La présente politique de confidentialité explique comment les informations personnelles sont collectées, utilisées et protégées.

Collecte des informations

Certaines informations peuvent être collectées lorsque les utilisateurs :

  • Remplissent un formulaire de contact
  • Envoient un message via le site
  • Contactent l’entreprise par email

Les informations collectées peuvent inclure :

  • Nom
  • Adresse email
  • Contenu du message

Ces données sont uniquement collectées dans le but de répondre aux demandes des utilisateurs.

Utilisation des données

Les données collectées sont utilisées uniquement pour :

  • Répondre aux demandes des visiteurs
  • Fournir des informations sur les services
  • Améliorer l’expérience utilisateur

Garoms-tech JBD-X s’engage à ne pas vendre, louer ou partager les données personnelles avec des tiers sans consentement.

Protection des données

Garoms-tech met en place des mesures techniques et organisationnelles pour protéger les informations personnelles.

Toutefois, aucune transmission sur internet ne peut être garantie totalement sécurisée.

Les utilisateurs reconnaissent utiliser internet à leurs propres risques.

Durée de conservation

Les données personnelles sont conservées uniquement pendant la durée nécessaire au traitement des demandes.

Une fois les demandes traitées, les informations peuvent être supprimées ou archivées de manière sécurisée.

Droits des utilisateurs

Les utilisateurs disposent de plusieurs droits concernant leurs données personnelles :

  • Droit d’accès
  • Droit de rectification
  • Droit de suppression

Toute demande concernant les données personnelles peut être envoyée à l’adresse suivante :

contactoptimistetech@gmail.com

Modification de la politique

Garoms-tech JBD-X peut modifier cette politique de confidentialité à tout moment afin de s’adapter aux évolutions légales ou techniques.

Les utilisateurs sont invités à consulter régulièrement cette page.
